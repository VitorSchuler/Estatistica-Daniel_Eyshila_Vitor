## 📌 Sobre o Projeto
Este repositório contém os scripts, dados e a documentação do trabalho final (3º Estágio) da disciplina de Estatística Aplicada. O objetivo principal é desenvolver uma Análise de Regressão Linear Múltipla completa, englobando desde a análise exploratória (EDA) até o ajuste, seleção do melhor modelo, validação de pressupostos e previsões.

## 📁 Estrutura do Repositório
* `dados/`: Contém a base de dados original fornecida para o trabalho e o dicionário de variáveis. *(Nota: A base original não deve ser alterada)*
* `scripts/`: Códigos de experimentação e funções auxiliares em R.
* `docs/`: Arquivo fonte Quarto (`.qmd`), arquivo de referências (`.bib`) e o relatório final renderizado em `.pdf`.

## 🛠️ Como Executar
Este projeto foi desenvolvido utilizando a linguagem **R** e o sistema de publicação **Quarto**. 

Para compilar o relatório:
1. Certifique-se de ter o [R](https://cran.r-project.org/) e o [RStudio](https://posit.co/download/rstudio-desktop/) instalados.
2. É necessário ter uma distribuição LaTeX instalada (recomenda-se o `tinytex` no R) para a geração do arquivo PDF.
3. Abra o projeto no RStudio, acesse o arquivo principal `.qmd` na pasta `docs/` e clique no botão **Render** (ou utilize o atalho `Ctrl+Shift+K`).
